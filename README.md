# Gravitational Wave Background Ceilings

Interactive visualization of astrophysical gravitational wave background ceilings based on energy reservoir constraints.

## Description

This Streamlit app visualizes the gravitational wave background (GWB) from various astrophysical sources:

- **SMBHBs**: Supermassive black hole binaries
- **IMBH**: Intermediate mass black hole - SMBH mergers
- **EMRI**: Extreme mass ratio inspirals
- **BNS**: Binary neutron stars
- **POPIII**: Population III black hole binaries
- **sBBHs**: Stellar-mass binary black holes

The app allows users to adjust mass density reservoirs and see how GWB amplitudes scale according to the Universal Scaling Law framework.
